# Code

This folder contains the main project code/configuration artifacts included with the final submission.

## `isaac_lab/`

Isaac Lab / RSL-RL configuration files used for Booster K1 full-body walking and walking-to-kicking policy development. The full working repository was developed on the team GPU workstation under the `booster_train` and Isaac Lab workspace.

## `mujoco/`

Reserved for MuJoCo-related scripts or configuration files. MuJoCo simulation outputs should be placed here if included.
