# Isaac Lab Code

This folder contains selected Isaac Lab task and PPO configuration files used for the final project.

Included files may include:
- Full-body K1 walking environment configuration.
- PPO/RSL-RL agent configuration.
- Supporting training assets and selected logs are stored under `final_report_assets/`.

The complete training workspace was developed on the team GPU machine using Isaac Lab and the `booster_train` task package.
